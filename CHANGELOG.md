## [1.0.0] - 2024-01-15

### Added
- 初始版本发布
- 支持 React 项目生成（16.x, 17.x, 18.x）
- 支持 Vue 项目生成（2.x, 3.x）
- TypeScript 支持
- React Router / Vue Router 集成
- qiankun 微前端支持
- 智能包管理器检测（npm/yarn/pnpm）
- ESLint 和 Jest 集成
- 完整的 webpack 配置
- 交互式 CLI 界面

### Technical
- 使用 TypeScript 编写
- 基于 Commander.js 的命令行界面
- Handlebars 模板引擎
- 完整的项目模板系统
